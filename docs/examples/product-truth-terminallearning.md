## Comment l'utiliser

1. Ouvre l'app **Getpostcraft**
2. Sidebar → **Settings**
3. Onglet **Comptes**
4. Section **Instagram** → trouve la carte `@terminallearning`
5. Sous le profil, tu verras la section **Product Truth** avec un textarea
6. Sélectionne tout ce qui est entre les ``` ``` ci-dessus (sans les backticks eux-mêmes)
7. Colle dans le textarea
8. Clique **Enregistrer**

À la prochaine génération de post, l'IA aura ce contexte injecté automatiquement →
plus de chiffre inventé, accents français appliqués, modules réels cités.
